Break-glass account rules
Create 2 cloud-only Global Admin accounts · Exclude from ALL CA policies · Store credentials in physical safe · Monitor with alert rule in Defender for any sign-in · Never use for daily tasks
Report-only monitoring (Entra ID → Sign-in logs)
Before flipping any policy ON, review Sign-in logs → filter "Conditional Access: Report-only" · Look for high-impact users hitting failures · Check for service accounts using legacy auth · Validate no business-critical app is being blocked
